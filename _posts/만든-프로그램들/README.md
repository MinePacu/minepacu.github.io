# 만든 프로그램들

이 폴더에는 직접 개발한 프로그램들의 개발 과정, 기능 구현 방법, 문제 해결 과정을 다룬 포스트들이 저장됩니다.

## 하위 카테고리

### CustomWindow/
- CustomWindow 프로그램 관련 개발 일지
- 기능 구현 과정
- 문제 해결 사례

### CraftPresence/
- CraftPresence 프로그램 관련 개발 일지
- 기능 구현 과정
- 문제 해결 사례

## 포스트 작성 가이드

### 파일명 규칙
```
YYYY-MM-DD-제목.md
```

### Front Matter 예시
```yaml
---
title: 포스트 제목
date: YYYY-MM-DD HH:MM:SS +0900
categories: [만든 프로그램들, CustomWindow]
tags: [tag1, tag2]
---
```

### 권장 태그
- `customwindow`, `craftpresence` - 프로그램 이름
- `development`, `debugging` - 개발/디버깅
- `feature-implementation` - 기능 구현
- `problem-solving` - 문제 해결
- `wpf`, `csharp`, `minecraft`, `discord-rpc` - 기술 스택
